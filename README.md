# urban-trajectory-prediction

![Python](https://img.shields.io/badge/Python-3.10%2B-6b46c1?style=flat-square)
![PyTorch](https://img.shields.io/badge/PyTorch-trajectory%20prediction-8b5cf6?style=flat-square)
![Research](https://img.shields.io/badge/Type-research%20code-a78bfa?style=flat-square)

Research code for urban trajectory prediction on road networks using next-node forecasting and full-path evaluation. The repository includes training pipelines, pretrained checkpoints, and k-fold results for LSTM, Transformer, GNN, and GraphIDyOM models across original, noisy, and mixed trajectory scenarios.
